Step-1: Install Required Libraries

Before running this project make sure you have all the required libraries installed in your system or in current environment.

for doing so , open this directory in terminal and run the below given command:

pip install -r requirements.txt

This will install all the libraries required for this project which don't come pre installed in python.



Step 2: Install Tesseract OCR Engine

Download the tesseract OCR engine from the link given below and install it in desired directory.

https://github.com/UB-Mannheim/tesseract/wiki



Step 3: Set path to tesseract.exe 

set the path to tesseract.exe according to the installation directory of OCR engine.

Open OCR_Engine.py file in a code editor and replace the previous given path with your path.


Step 3: Run the main.py file.
